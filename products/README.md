## 接入要求

- 在各自仓库根目录维护 `CLAUDE.md` 和 `README.md`
- 对外 API 遵循 OpenAPI 3.x 规范
- 内部通信使用 gRPC，Proto 定义提交至 `api/proto/`
- 通过 APISIX 网关统一暴露外部接口
